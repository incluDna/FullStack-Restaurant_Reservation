## Prerequisites

Requires [Node.js](https://nodejs.org/en) installed, including npm.

## Getting Started

- Create a `config.env` file. Here is an example:

```
PORT=5000
NODE_ENV=development

MONGO_URI=

JWT_SECRET=
JWT_EXPIRE=
JWT_COOKIE_EXPIRE=
```

- Then, initialize with

```
npm install
```

- Start a development server with

```
npm run dev
```
